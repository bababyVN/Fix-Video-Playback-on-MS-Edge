## 🛑 The Problem
Are you experiencing these issues when using Microsoft Edge on a standard office laptop (without a dedicated graphics card)?
* YouTube videos stutter, drop frames, or feel laggy compared to Google Chrome.
* Colors on Netflix or other streaming sites look washed out, pale, or overly grainy.
* Your laptop gets hot when watching high-res videos.

## ✅ The Solution
This repository contains a simple, one-click script (`Optimize-Edge.bat`) that automatically configures Microsoft Edge's hidden settings to run perfectly on integrated graphics. No coding knowledge required!

## 📥 How to Use (Step-by-Step)
1. Download the `Optimize-Edge.bat` file to your computer.
2. Double-click the file to run it.
3. *(Note: If Windows shows a blue "Windows protected your PC" popup, click **More info**, then click **Run anyway**. This happens because the script is downloaded from the internet).*
4. A black window will appear for a few seconds. It will safely close Microsoft Edge, apply the fixes, and restart the browser automatically.
5. Enjoy smooth, lag-free videos with vibrant colors!

## 🛡️ Is it Safe? (How to Undo)
**Yes, 100% safe.** This script does NOT install any software, and it does NOT touch your Windows registry. It only changes 6 standard toggles inside Edge's hidden settings (`edge://flags`).

**Want to go back to normal? It's extremely easy:**
1. Open Microsoft Edge.
2. Type `edge://flags` into the address bar and press Enter.
3. Click the **"Reset all"** button at the top right corner.
4. Click **Restart**. Everything is back to factory settings!
